# Zavia-AI v2

Enterprise AI engineering studio website — built with Next.js 14, Tailwind CSS, and Framer Motion.

## Stack

- **Next.js 14** (App Router, SSR/SSG)
- **Tailwind CSS** (dark cinema theme)
- **Framer Motion** (scroll animations, page transitions)
- **TypeScript**
- **Deployed on Vercel**

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Deploy on Vercel

Push to GitHub → Import project on [vercel.com](https://vercel.com) → Deploy.

## Pages

- `/` — Homepage
- `/about` — About, mission, offices, timeline
- `/services` — Five service offerings
- `/projects` — Case studies and project portfolio
- `/architecture` — Six-layer reference architecture
- `/industries` — Six target industries
- `/team` — Team profiles
- `/contact` — Contact form with intent routing
